# Stock Market Dashboard

This project analyses stock price performance among major technology companies.

The companies analysed include:

Apple (AAPL)
Microsoft (MSFT)
Amazon (AMZN)
Google (GOOGL)

Financial data was collected using the Python library yfinance, which retrieves historical stock market data from Yahoo Finance.

The project includes:

• stock price comparison
• financial risk (volatility) analysis
• data visualisation using Plotly

The goal of the dashboard is to demonstrate how Python can be used to analyse financial data and visualise market trends.
